# -ARDUINO---DATA_Rx_Tx_MEGA2650

Goal : 
Transmit Datas Via AUDIO channel of FPV VTx to control Antenna Traker

State : 
[In PROGRESS]

Require :
1 x GPS Module ( NEO6M W/O Compass )
2 x Arduino MEGA2650
1 x VRx TS351
1 x VRx RC805
IDE Tiny GPS++ ( To be tested )
IDE Wire Lib
IDE Arduino 1.6.0

- Master :
SDA ( pin 20 ) / SCL ( pin 21 )

- Slave :
SDA ( pin 20 ) / SCL ( pin 21 )
